# ApproveIt — Frontend

Next.js (App Router) frontend for the ApproveIt expense management system.

## Tech stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Connects to
Spring Boot backend at `http://localhost:8080` (see `../backend`).
